# tripeaks
A simple [Tri Peaks solitaire](https://en.wikipedia.org/wiki/Tri_Peaks_(game)) in Python + PyGame

## Controls
- **Left Mouse Button** — Draw a card from stock or take a card from pyramids
- **R** — Start a new game
- **Z** — Revert up to 3 actions

## How to run
`python tripeaks.py`
